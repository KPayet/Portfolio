Prédictions Click-Through-Rate Criteo
  • Machine Learning
  • Big Data
  • Python
  • R
  • Spark

Cette page décrit mon travail sur le challenge Criteo, hébergé par Kaggle il y a un peu plus d'un an. C'est un jeu de données très intéressant, car celles-ci peuvent être délicates à traiter, en raison de la large quantité de mémoire requise. C'est l'une des raisons qui m'ont poussé à m'intéresser à ce projet. Le reste de cette page présente ma manière d'aborder ce problème. Le but de ce challenge était de produire un modèle de Machine Learning pour la prédiction du "Click-Through-Rate" (CTR), en utilisant un ensemble de features fournies par Criteo...
